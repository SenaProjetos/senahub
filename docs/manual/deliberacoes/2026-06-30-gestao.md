# Deliberação — Seção Gestão

- **Data:** 2026-06-30
- **Funcionalidades:** `/juridico`, `/licitacoes`, `/qualidade`, `/patrimonio`,
  `/patrimonio/ti`.

## Participantes
Presidente, Iniciante, Experiente, UX, Backend, Frontend, QA, Negócios, Diretor,
Treinamento, Revisor Técnico, Arquiteto, Product Owner, Suporte.

## Descobertas (inspeção de código)
- **Licitações** (`app/.../licitacoes/page.tsx`): `licitacoes:ver`/`gerir`; painel +
  lista com filtros status (múltiplo)/órgão/q + paginação. Parâmetros gerais
  (`/configuracoes/licitacoes`) = admin (revisao-telas 0.A.5).
- **Jurídico** (`app/.../juridico/page.tsx`): `juridico:ver`/`gerir`. Lê
  `DocumentoJuridico` (pastas, versões, **aceites com hash**), `Certidao` (tipo,
  **validade**), `ModeloContrato`. **Funcional e rico** — diverge do CLAUDE.md, que diz
  "juridico é placeholder (actions.ts only)".
- **Qualidade** (`app/.../qualidade/page.tsx`): `qualidade:ver` (nav: admin/supervisor).
  Índice de **retrabalho** (% disciplinas ativas com revisão), SLA de entregas, revisões
  por disciplina, **snapshot mensal** (dia 1º).
- **Patrimônio** (`app/.../patrimonio/page.tsx`): `patrimonio:ver`/`gerir`; ativos por
  categoria, alocados a colaboradores.
- **TI** (`app/.../patrimonio/ti/page.tsx`): `patrimonio:ti` (papel ti + admin/supervisor);
  máquinas + alocação + ativos sem máquina.

## Questionamentos
- **Divergência CLAUDE.md × código (jurídico).** CLAUDE.md trata jurídico como placeholder;
  a rota é plenamente funcional. → Documentado o **comportamento real**; registrada
  pendência para atualizar o CLAUDE.md.
- Qualidade é admin/supervisor only? → conforme nav-config (`roles: ["admin","supervisor"]`).

## Opiniões dos Especialistas
- **Backend:** confirmou que a lógica do jurídico está inline na página + componentes
  (por isso o módulo parece "fino"), mas a feature existe.
- **QA:** índice de retrabalho precisa do "menor é melhor" explícito — incluído com as
  faixas de cor.
- **UX:** distinguir Patrimônio × TI para o usuário — FAQ em ambos.
- **Negócios:** validade de certidão é crítico (habilitação em licitação) — referência
  cruzada Jurídico↔Licitações.

## Discussão
Consenso: documentar comportamento real (não o CLAUDE.md). Uma página por funcionalidade,
com referências cruzadas Jurídico↔Licitações e Patrimônio↔TI.

## Divergências
- **CLAUDE.md vs. código (jurídico):** prevalece o **código**. Decisão registrada como
  pendência de atualização da doc de desenvolvimento.
